Odoo 8 Point of Sale Extension

Return products from Point of Sale Frontend

How to use

  • User has to select pos order which is created from pos, and can return products during next purchase and can be managed from frontend also.
  • First we have created normal pos order and purchased different quantity products.


  • And after completion of order, we have displayed barcode so user can use during return. This barcode is the number of order only.


  • This is the number which can be scanned from barcode.


  • Now, when we click on "Return Order" button, then there will be prompt to enter order number, which can be scanned from receipt also.


  • Automatically, we can see, all products according to scanned order will be visible on screen with quantity.


  • User can not return more than purchased quantity.


  • During return, if we select "Select Mode" then we can do normal sale also.
